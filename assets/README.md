# Glass-Box Assets

This directory contains visual assets for the Glass-Box skill.

## Icon

The Glass-Box icon represents:
- **Transparency**: Clear view into agent operations
- **Memory**: Persistent storage across sessions
- **Bidirectionality**: Two-way arrow (query ↔ write)

## Usage

These assets can be used in:
- Documentation
- GitHub repository
- Skill packages
- Web interfaces
